# GLM_quota_kicker

> 智谱 AI API 配额自动唤醒工具

自动在配额重置后触发 API 调用，最大化利用你的智谱 AI API 配额。

## 📖 简介

智谱 AI 的 API 配额每 5 小时重置一次（00:00 → 05:00 → 10:00 → 15:00 → 20:00）。如果配额重置后不立即使用，新配额会在下一个周期到来时作废。

本工具通过定时任务自动在配额重置后发送简短请求，确保每一份配额都被充分利用。

### ✨ 特性

- 📊 **智能配额管理** - 零消耗查询配额，自动判断激活/调度
- 🧠 **智能检测** - 唤醒前自动检查配额，避免浪费
- 🎯 **自动重试** - 遇到配额不足时自动解析重置时间并安排重试
- 🎲 **随机消息** - 支持自定义消息列表，随机选择避免被识别为异常
- 📝 **详细日志** - 完整的运行日志记录
- 🔧 **简单配置** - 交互式配置向导，一键完成设置
- 🖥️ **跨平台** - 支持 macOS 和 Linux

## 📋 系统要求

- **操作系统**: macOS 或 Linux
- **依赖工具**:
  - `bash` 4.0+
  - `jq` (JSON 解析工具)
  - `curl` (HTTP 客户端)

```bash

# Ubuntu/Debian
sudo apt install jq

```

## 🚀 快速开始

### 1. 获取项目

```bash
git clone https://github.com/ssdiwu/GLM_quota_kicker.git
cd GLM_quota_kicker
```

### 2. 配置

```bash
./wake.sh -s
```

按提示输入 API Key 即可。

## ⚙️ 配置文件

### 配置文件结构

```json
{
  "api": {
    "key": "your-api-key-here",
    "model": "glm-4.7"
  },
  "provider": {
    "name": "智谱 AI",
    "baseUrl": "https://open.bigmodel.cn/api/anthropic",
    "endpoint": "/v1/messages",
    "authHeader": "x-api-key"
  },
  "request": {
    "prompts": [
      "hi",
      "你好",
      "hello"
    ]
  }
}
```

### 配置说明

| 字段 | 说明 | 示例 |
|------|------|------|
| `api.key` | 智谱 AI API Key | `your-api-key-here` |
| `api.model` | 使用的模型 | `glm-4.7`, `glm-4-flash` |
| `request.prompts` | 唤醒消息列表（可选） | 见上方示例 |

## 🔧 命令选项

```bash
./wake.sh [选项]

选项:
    -h              显示帮助信息
    -q              智能配额管理（自动判断激活/调度）
    -v              显示版本信息
    -a [CMD]        防睡眠控制（仅 macOS）
                    CMD: start|stop|status|restart
    -d TIME         后台定时模式（高级用法）
    -s              重新配置
```

### 使用示例

```bash
./wake.sh -q               # 智能配额管理（推荐）
./wake.sh                  # 直接唤醒
./wake.sh -d 1514          # 后台定时执行（高级用法）
./wake.sh -s               # 重新配置
```

### 内置定时器功能（高级用法）

```bash
# 后台定时执行
./wake.sh -d 15:14
```

**支持的时间格式**：
- `HH:MM` - 如 `15:14`
- `HHMM` - 如 `1514`、`0600`
- `HH:MM:SS` - 如 `15:14:30`
- `HHMMSS` - 如 `151430`

### 自动重试功能

当配额不足时，脚本会自动检测重置时间并安排重试：

```
执行 → 配额不足 → 解析重置时间 → 创建后台重试 → 自动唤醒
```

无需任何手动操作。

## 💤 系统睡眠建议

### 运行状态说明

| 系统状态 | 定时任务是否运行 | 说明 |
|---------|-----------------|------|
| ✅ 屏幕关闭 | ✅ 正常运行 | 后台进程不受影响 |
| ❌ 电脑睡眠 | ❌ 不运行 | 需要保持唤醒状态 |
| ❌ 合盖 | ❌ 不运行 | Mac 会睡眠 |

### MacBook 用户

**防睡眠功能**：

```bash
# 启动防睡眠（晚上睡觉前）
./wake.sh -a

# 停止防睡眠（第二天早上）
./wake.sh -a stop
```

**组合使用**：

```bash
# 启动防睡眠 + 后台定时
./wake.sh -a -d 15:14
```

## ❓ 常见问题

### Q: 为什么需要这个工具？

A: 智谱 AI 配额每 5 小时重置，不使用的话新配额会在下一个周期作废。

### Q: 会消耗很多配额吗？

A: 不会。每次只发送极简请求（约 6 tokens）。

### Q: 如何更改 API Key？

A: 运行 `./wake.sh -s` 重新配置。

### Q: API Key 存储安全吗？

A: 配置文件在本地，不会上传。`.gitignore` 已配置忽略。

### Q: 支持 Windows 吗？

A: 目前仅支持 macOS 和 Linux。Windows 可使用 WSL。
